# Agent Images

Place agent images/logos in this folder with the following filenames:

- `ark-invest.png` - ARK Invest logo for "ARKian Visionary"
- `cramer.png` - Jim Cramer/Mad Money logo for "Cramer Cultist"
- `big-short.png` - Logo/icon for "The Big Short Guy"
- `wsb.png` - WallStreetBets logo/icon for "WSB Degenerate"
- `buffett.png` - Warren Buffett/Berkshire Hathaway logo for "The Warren Buffett Disciple"

## Image Requirements

- Format: PNG (preferred) or SVG
- Size: Recommended 128x128px or larger (will be displayed as 32x32px in leaderboard, 40x40px in detail view)
- Style: Square images work best as they'll be displayed in circular frames
- Background: Transparent backgrounds preferred

## Where to Find Images

You can search for these images online:
- ARK Invest logo: Search "ARK Invest logo PNG"
- Jim Cramer: Search "Jim Cramer Mad Money logo PNG"
- The Big Short: Search "The Big Short movie logo PNG" or use a bear/bubble icon
- WallStreetBets: Search "WallStreetBets logo PNG" or Reddit WSB icon
- Warren Buffett: Search "Warren Buffett Berkshire Hathaway logo PNG"

Make sure you have the rights to use these images or use royalty-free alternatives.
